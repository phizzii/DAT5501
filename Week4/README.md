## About The Code

Fourth week of content includes a script cleaning two datasets (number of deaths by neurological and mental disorders and the annual working hours in both the UK and China), and plotting graphs to analyse the trend in both datasets and then the trend between them in preparation for our group presentation to the class. This was then tested using a testing suite separate to this script `[see test_uk_vs_china_graphs.py in tests folder]`
