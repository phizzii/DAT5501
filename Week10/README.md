## About The Code

Tenth week of content includes cleaning a dataset from the UV Irvine Machine Learning Repository, completing model selection and training followed by model evaluation and optimisation by testing on test data and looking at performance. This is done by utilising a decision tree on a dataset with a labelled dataset with a discrete outcome then testing it using a testing suite separate to this script `[see test_machine_learning.py in tests folder]`
