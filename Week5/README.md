## About The Code

Fifth week of content includes a duration calculator which calculates the number of days passed since a date inputted by the user to today. This also includes cleaning a dataset of 1 year of historical asset price on a company I have chosen (Rocket Lab), plotting the closing price vs the date, claculating the daily percentage change and claculating the standard deviation of percent daily changes. Finally, this folder additionally contains a script which plots a histogram of US election data of the fraction of votes as well as a plot comparing the vote fraction between two candidates of my choice (Bernie Sanders and Ted Cruz) then testing all scripts using a testing suite separate to this script `[see test_asset_prices.py, test_duration_cal.py and test_us_election.py in tests folder]`
