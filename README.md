## About The Project

This repo has all the code I have written for my DAT5501 data analysis module for Year 2 university semester A. Reinforcing my learning of using Python library modules such as Pandas, MatPlotLib and other modules. This module has also supported my upskilling in creating testing suites for my scripts linking CircleCI so when changes are pushed, I can see whether or not my tests that I have written for each piece of code has passed or not.
