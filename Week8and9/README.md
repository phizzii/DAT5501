## About The Code

Eighth and ninth week of content includes one script to calculate the time taken to sort 1 years worth of daily changes in asset price for Rocket Lab for each sorting algorithm (Bubble sort, Insertion sort, Merge sort and Quick sort) and plotting graphs on the Big O time complexity of each. An additional script for fitting the sub-sample of a dataset (global) of my choice (cherry tree peak day of blossom) for all but the last ten years and then using each polynomial fit (for orders up to 9) forecasting each polynomial 10 years into the future and comparing the result with the actual data to find the best suited model. These were then tested using a testing suite separate to this script `[see test_daily_price_change_sorting.py and test_fitting_forecasting.py in tests folder]`
